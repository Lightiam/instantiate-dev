Deployment Instructions

Step 1: Deploy to Netlify

Click the deploy button above to automatically:

  • Create Netlify site from your GitHub repository
  • Configure build settings (npm run build)
  • Set up continuous deployment

Step 2: Configure Custom Domain

In Netlify dashboard:

  1. Go to Domain settings
  2. Add custom domain: instantiate.dev
  3. SSL certificate will be auto-provisioned

Step 3: Environment Variables

Add these in Netlify Environment Variables:

  • GROQ_API_KEY - Your AI chat API key
  • DATABASE_URL - PostgreSQL database URL
